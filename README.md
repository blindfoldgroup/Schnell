# Schnell

## on Linux

### Add 'Open in Schnell'

```
wget -qO- https://raw.githubusercontent.com/blindfoldgroup/Schnell/master/install.sh | bash
```

### Remove 'Open in Schnell'

```
rm -f ~/.local/share/nautilus-python/extensions/schnell-nautilus.py

```
